# 📸 AI Photo Editor (OpenCV + Streamlit)

## 🚀 Overview
This project is a web-based Photo Editor built using OpenCV and Streamlit. It allows users to upload an image and apply various transformations and filters in real time.

## ✨ Features
- Image Upload
- Resize, Brightness & Contrast Adjustment
- Filters:
  - Grayscale
  - Blur
  - Sharpen
  - Warm Effect
  - Portrait Background Blur
- Download Edited Image

## 🔧 Tech Stack
- Python
- OpenCV
- Streamlit
- NumPy

## ▶️ Run Locally
```bash
pip install -r requirements.txt
streamlit run app.py
```
